# Alsa capture visualization for console

## Requirements

> sudo apt install libasound2 python-alsaaudio python3.5 python3-posix-ipc python3-pip python3.5-dev
> sudo pip3 install pyalsaaudio
> sudo pip3 install posix_ipc

# run

> chmod +x ./check_audio_level.py 
> ./check_audio_level.py --help

example:

> ./check_audio_level.py --device default --rate 16000
